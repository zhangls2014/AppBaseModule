# BaseUtils
自己在开发过程中经常使用的工具/代码片段

#### base module 包含的内容

- retrofit 的封装
- 自定义 View 相关的扩展方法
- 一些常见的扩展方法
- 简单的对话框实现

#### 使用方法

[![](https://jitpack.io/v/zhangls2014/BaseUtils.svg)](https://jitpack.io/#zhangls2014/BaseUtils)

```groovy
allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}

dependencies {
  implementation 'com.github.zhangls2014:BaseUtils:Tag'
}
```
